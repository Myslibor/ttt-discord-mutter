# ttt-discord-mutter
A TTT Gmod addon and Discord bot, that mutes you when you die.

Requires https://github.com/timschumi/gmod-chttp

After firts lunch a config file will be created. Paste your Server ID and Bot Token in there.

In discord:
!tttbot_map {your STEAMID64} - maps your discord account to STEAMID64, can be eddited in id_map.json (creates it self after first lunch)

!tttbot_end - stops the bot
